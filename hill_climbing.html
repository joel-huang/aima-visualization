the
hill climbing
algorithm

Hill climbing is a greedy algorithm that compares its
neighboring values, checks which is higher, and moves there.
Click on a spot to start from, and then start climbing!
Hit on new hills to make new dunes. 

hill_climb():
    current = start_point
    while true:
        neighbor // a highest-valued successor of current
        if neighbor.value <= current.value:
            return current // we found it!
        current = neighbor
